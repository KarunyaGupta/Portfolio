# ashthetic-portfolio-final

Personal portfolio website built with React and Vite.

## Overview

This repository contains a responsive portfolio site implemented with React (v18) and Vite. It includes pages for Home, About, Projects, Blog, Gallery, Certificates, Contact, and Resume, plus shared components and CSS modules.

## Tech Stack

- React
- Vite
- Framer Motion
- React Router DOM
- D3 (for visualizations)

## Prerequisites

- Node.js (v16+ recommended)
- npm (or yarn)

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build locally:

```bash
npm run preview
```

## npm Scripts

- `dev`: Run the Vite dev server (`vite`).
- `build`: Create a production build (`vite build`).
- `preview`: Preview the production build (`vite preview`).

- Add new pages under `src/pages` and register routes in `src/App.jsx`.
- Place shared components in `src/components` and import them into pages.
- Use `public/` for static assets referenced directly by URL.

## Deployment

This site can be deployed to static hosting providers like Netlify, Vercel, or GitHub Pages. Build the app with `npm run build` and deploy the contents of the `dist/` folder (Vite output).
